## About & Instructions:

This mod is a small story mod where you take the place of an ordinary void sprite. Spend your time munching on asteroids and ships, or enjoying the small (but expanding) storyline between you and other species. Navigate the hardships of the language barrier, and get rewarded with different colors!

First, make sure that you start the game with a new save, under the "Void Sprite" start. To progress the story, land to and from Nasqueron to trigger missions (most occur within 2-4 landings). Remember to "clean up" mission targets by eating disabled ships (they're tasty, I promise), which will allow the mission to be marked as complete. The current end to the storyline is after the second battle, although this is to be expanded.
