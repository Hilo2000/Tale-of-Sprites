## About & Instructions:

This mod is a small story mod where you take the place of an ordinary void sprite. Spend your time munching on asteroids and ships, or enjoying the small (but expanding) storyline between you and other species. Navigate the hardships of the language barrier, and get rewarded with different colors!

To start the plugin, make sure that you make a new save under the "Void Sprite" start. To progress the story, land to and from Nasqueron or Slylandro to trigger missions (the first will start randomly, then the following will happen within 4 landings or by eating certain asteroids). Remember to "clean up" mission targets by eating disabled ships (they're tasty, I promise), which will allow the mission to be marked as complete. The main storyline is compelete, but several features and a planned alternate storyline have not yet been released.
